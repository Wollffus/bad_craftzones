## Requirements

# How to install

- unzip the folder
- drag `bad_craftzones` into your resources folder
- Change the config.txt to config.lua or the script will not work
- Open the config.lua and change to what you like
- ensure `bad_craftzones` in your server.cfg
- restart your server
